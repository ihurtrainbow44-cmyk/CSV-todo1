# ToDo App (C++)

## Описание
Консольное приложение для управления задачами.

## Функции
- Добавление задач
- Завершение задач
- Импорт/экспорт CSV

## Технологии
- C++
- CMake
- SQL Server

## Запуск
1. Открыть проект в Visual Studio
2. Запустить

## Тестирование
Реализован тест для проверки CSV импорта/экспорта
